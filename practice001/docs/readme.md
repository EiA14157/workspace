this is project to compare recursive functions and iterative functions
recursive.h file provides sturctures of functions which solve fibonacci and hanoi tower probloem in recursive and iterative ways
recursive.c file provides actual functions using structures from recursive.h
main.c executes those functions and compares runtime.

Parameter n should not be over 1000.
This limit can be controled on recursive.c
